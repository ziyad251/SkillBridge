# SkillBridge

SkillBridge is a full-stack Learning Management System (LMS) that enables instructors to create and manage courses while allowing students to enroll, purchase, and learn through an intuitive platform.

## Features

- User Authentication (JWT)
- Role-based Access (Admin, Instructor, Student)
- Course Creation & Management
- Student Enrollment
- Secure Course Purchase using Razorpay
- Review & Rating System
- Certificate Generation
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Razorpay SDK
